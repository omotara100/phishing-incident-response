# Phishing Incident Detection

## Incident Overview
A user reported a suspicious email claiming to be from IT Support requesting an immediate password reset.

## Detection Source
- User-reported email
- Email security alert (simulated)

## Suspicious Indicators
- Sender domain does not match corporate domain
- Urgent language pressuring the user to act
- Embedded link requesting credential input
- Generic greeting instead of user name

## Initial Assessment
Based on the indicators observed, the email is suspected to be a phishing attempt designed to harvest user credentials.
